# Zain Ahmad — Portfolio 2026

Personal portfolio showcasing UX/UI design, multimedia work, and creative development.

## Tech Stack

- **HTML / CSS / JavaScript** — No frameworks, fully hand coded
- **GSAP 3.12.5 + ScrollTrigger** — Scroll driven animations
- **Lenis 1.1.18** — Smooth scrolling
- **Horizon & SF Pro Display** — Custom typography

## Getting Started

No build step required. Open `index.html` in a browser or serve with any static file server:

```bash
# Using Python
python -m http.server 8000

# Using Node
npx serve .
```
