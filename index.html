Featured Case Study CIS 2170 — Team of 5

HARVEST HANDS

Inventory management mobile application designed for food bank operations

RoleLead UI Designer
Team5 Members
DurationSemester Project
ToolsFigma
EDITOR CREATOR

THE CHALLENGE

Food banks need an efficient way to track inventory, manage item approvals, and handle real time notifications for stock levels. The existing process was manual and prone to errors, requiring a streamlined digital solution.

MY CONTRIBUTION

As lead UI designer, I designed the complete visual system: wireframes, all screens, the design system (dark theme with color coded status indicators), and ensured consistency across every user flow.

DESIGN SYSTEM

Created a comprehensive dark theme design system with color coded status indicators, consistent typography, and reusable components that enabled the team to build screens rapidly while maintaining visual cohesion.

USER FLOWS

Mapped every user journey from login to checkout, ensuring intuitive navigation and minimal friction. Created detailed wireframes for approval workflows, stock alerts, and admin controls.

ACCESSIBILITY

Implemented WCAG compliant color contrast ratios and touch targets. Designed responsive layouts that adapted seamlessly from mobile to tablet, prioritizing one handed usability for warehouse environments.

OUTCOME

Delivered a polished, fully interactive Figma prototype that streamlined food bank inventory workflows. The design reduced manual entry time and provided real time visibility into stock levels and approval statuses.
